# go-auth

My client authentication libraries for Golang.


## Libraries

- LDAP
- OAuth/OIDC 2.0
- Linux PAM
- PKCS #11
- RADIUS
- TACACS+
- TPM 2.0
- Yubikey


## Facade

The [api package](pkg/api/) contains a factory pattern for working with all of the clients through a uniform constructor.
